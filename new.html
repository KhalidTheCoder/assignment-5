Question-1: What are the different ways to select an element in the DOM?

The different ways to select a DOM element are getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML changes the HTML, innerText shows only visible text, and textContent gets all the text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent handles the events for its child elements by using a single event listener.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts from the target element and moves up to its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Use document.createElement() to create, appendChild() or append() to add, and removeChild() or remove() to remove elements.
